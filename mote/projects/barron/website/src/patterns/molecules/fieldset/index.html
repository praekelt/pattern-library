Fieldset

This is the molecule is used to construct a group of option buttons. The Block level class is OptionButtonGroup which is present on the container, this has a modifiers Fieldset--inverted for dark backgrounds and Fieldset--nested for nested Fieldsets. This is used in conjuction with Element classes on the item fields Fieldset-legend and Fieldset-descriptionHeading for descriptive text headings.
