Field Label

This is the atom for an <label> tag. The Block level class is FieldLabel which can be used in conjuction with the class modiefier FieldLabel--inverted when the input is present on the dark background. 
            
                
<!-- field label -->
<label class="FieldLabel" >label text goes here</label>
<!-- field label inverted -->
<label class="FieldLabel FieldLabel--inverted" >label text goes here</label>
