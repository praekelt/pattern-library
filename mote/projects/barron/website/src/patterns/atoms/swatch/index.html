Colour Swatch

This is the atom is used to construct a colour swatch. The Block level class is Swatch which is present on the container, this has a modifier Swatch--inverted for dark backgrounds. This is used in conjuction with Element classes on the input fields Swatch-input and the label field Swatch-label and the nested <span> in the label Swatch-checkBox. 
            
                
<!-- radio option buttons -->
<div class="Swatch">
    <input class="Swatch-input" type="checkbox" name="thing1" value="1" id="thing1">
    <label class="Swatch-label" for="thing1"><span class="Swatch-checkBox"></span></label>
</div>
<!-- inverted option buttons -->
<div class="Swatch Swatch--inverted">
    <input class="Swatch-input" type="checkbox" name="thing1" value="1" id="thing1">
    <label class="Swatch-label" for="thing1"><span class="Swatch-checkBox"></span></label>
</div>
