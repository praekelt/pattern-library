Field Input

This is the atom for an <input> tag. The Block level class is FieldInput which can be used in conjuction with the class modiefier FieldInput--inverted when the input is present on the dark background. 
            
                
<!-- input field -->
<input class="FieldInput" />
<!-- input field inverted -->
<input class="FieldInput FieldInput--inverted" />
