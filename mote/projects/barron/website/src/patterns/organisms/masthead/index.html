Masthead

The masthead contains the Primary Nav, Logo, Search and Menu Carousel. It is very important to note that by default the Primary Nav contains only a single link ("Menu") which anchors to the #menu-fallback DOM node, which is the Fallback Nav. Once the JS kicks in, the Menu Carousel items will be contructed in their entirety on the client-side based on the structure of the Fallback Nav.
